# 🤖 AI Chatbot — Ollama + Groq

A streaming ChatGPT-style chatbot built with Python and Streamlit, supporting both **local LLMs via Ollama** and **cloud LLMs via Groq API**.

🔗 **[Live Demo](YOUR_STREAMLIT_URL_HERE)** — update this after deploying

---

## ✨ Features
- 💬 Real-time streaming responses
- 🧠 Conversation memory
- 🌡️ Temperature control
- ☁️ Groq Cloud mode (Llama 3, Mistral, Gemma)
- 💻 Local Ollama mode
- 🔧 Custom system prompt

## 🛠️ Tech Stack
Python · Streamlit · Groq API · Ollama

## 🚀 Run Locally
```bash
git clone https://github.com/vamshi1823/clone-chatgpt.git
cd clone-chatgpt
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
streamlit run app.py
```

Get a free Groq API key at [console.groq.com](https://console.groq.com)
